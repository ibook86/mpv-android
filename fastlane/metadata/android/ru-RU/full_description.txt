mpv-android - это видеоплеер для Android, основанный на <a href="https://github.com/mpv-player/mpv">libmpv</a>.

Особенности:
* Аппаратное и программное декодирование видео
* Поиск с помощью жестов, управление громкостью/яркостью и многое другое
* Поддержка libass для стилизованных субтитров
* Поддержка дополнительных (или двойных) субтитров
* Расширенные настройки видео (интерполяция, дебандинг, масштабирование, ...)
* Воспроизведение сетевых потоков с помощью функции "Открыть URL"
* Поддерживается фоновое воспроизведение, картинка в картинке, ввод с клавиатуры
